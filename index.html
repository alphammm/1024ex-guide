REFERENCE

核心术语速查

掌握这些概念，让你的交易更加得心应手

永续合约 Perpetual
没有到期日的合约，可以一直持有。通过资金费率机制锚定现货价格
杠杆 Leverage
放大交易仓位的工具。10x 杠杆 = 用 100U 保证金控制 1000U 的仓位
保证金 Margin
开仓时需要抵押的资金。保证金不足时会触发强制平仓（爆仓）
强平 Liquidation
当亏损达到一定比例，系统自动平掉仓位以防止进一步亏损
资金费率 Funding Rate
多空双方定期支付的费用，用于使合约价格趋近现货价格
止盈/止损 TP/SL
Take Profit / Stop Loss，预设价格自动平仓，控制风险和锁定利润
滑点 Slippage
实际成交价格与预期价格之间的差异，流动性越好滑点越低
Gas 费
在区块链上执行交易需要支付的网络手续费，Arbitrum 链上 Gas 费极低
FAQ

常见问题

新手最常遇到的问题，这里都有答案

1024EX 需要 KYC（身份验证）吗？
不需要。1024EX 是完全去中心化的交易所，连接钱包即可使用所有功能，无需提交任何个人信息或进行身份验证。
支持哪些链的资产？
1024EX 主要运行在 Arbitrum 链上。你可以通过内置的 Bridge 功能从 Ethereum、BSC、Polygon 等主流链跨链桥接资产到 Arbitrum。
交易手续费是多少？
手续费根据交易对和订单类型有所不同。一般市价单（Taker）手续费略高于限价单（Maker）。具体费率请参考平台的 Fee Schedule 页面。持有积分或推荐活跃用户可享受手续费折扣。
我的资金安全吗？会被黑客攻击吗？
1024EX 是去中心化协议，资金由智能合约托管，非人工管理。所有交易在链上公开透明可查。但请注意：DeFi 协议本身仍存在智能合约风险，建议只投入你可以承受损失的资金。
新手应该用多大的杠杆？
强烈建议新手从 2x - 5x 杠杆开始。高杠杆意味着高风险，10x 杠杆下价格反向波动 10% 就会触发强平。先熟悉市场节奏，再逐步调整杠杆策略。
如何使用 API 进行量化交易？
进入账户设置页面，在 API 管理中创建 API Key。1024EX 提供 REST API 和 WebSocket 接口，支持市场数据查询、订单管理、仓位管理等完整功能。详细文档可在平台的 API Docs 页面查阅。

风险提示

  • 永续合约交易具有高风险，杠杆会同时放大收益和亏损，请务必充分了解风险后再参与
  • 只使用你能承受完全损失的资金进行交易，切勿借贷或使用生活必需资金
  • DeFi 协议存在智能合约漏洞风险，过往安全记录不代表未来安全保障
  • 加密货币市场波动剧烈，价格可能在短时间内大幅变动
  • 新手务必先使用小额资金熟悉平台操作，设置好止损后再逐步加大投入
