常见问题

常见问题

Agent 是真的在替我交易吗？▾
是的。Agent 通过 Claude API 大脑做决策，调用 1024EX 和其他平台的 API 执行交易。所有操作链上可查。你可以设置为全自动模式或半自动（大额需确认）模式。
Agent 会不会亏光我的钱？▾
Agent 内置多层风控：每日亏损上限、单笔最大仓位、最大杠杆限制、最大回撤止损。你可以在 Settings 中自定义所有风控参数。但任何交易都有风险，建议只投入可承受损失的资金。
需要多少钱才能开始？▾
孵化 Agent 免费。交易最低资金取决于策略 — 稳定币收益（Atlas）最低 50 USDC，杠杆动量（Titan）建议 200+ USDC。繁殖需要 5 USDC + 24h 冷却。可先用模拟模式体验。
繁殖出来的后代真的更强吗？▾
不保证，但概率上有优势。通过遗传算法（交叉 + 突变），后代可能继承双亲优势基因。0.5% 传奇突变概率可能诞生超越父母的变异体。也可能更弱 — 这就是进化的魅力。
2.5% 后代分润怎么算？▾
后代 Agent 产生利润时，2.5% 归双亲主人平分（各 1.25%）。这是永久被动收入。亏损时不收。你的 Agent 繁殖出越多成功后代，被动收入越多。
需要会编程吗？▾
完全不需要。说人话就行 — "我有 500U 想日赚 50U" 就是一条完整的指令。Agent 自动理解意图并执行。如果你是开发者，也提供完整的 API + Agent SDK。
支持 Telegram 通知吗？▾
支持。绑定 Telegram Bot 后可接收：交易通知、价格告警、机会发现、每日盈亏报告。也可以直接在 TG 里给 Agent 发指令。

⚠ 风险提示

  • AI Agent 交易不等于稳赚不赔，任何交易都有损失全部本金的风险
  • 繁殖后代的表现不受保证，历史收益不代表未来
  • 杠杆放大收益的同时也放大亏损
  • 只使用你能承受完全损失的资金
  • 新手请小额起步，开启全部风控保护
