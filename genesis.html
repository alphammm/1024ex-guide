FAQ

常见问题

什么是 Agent 基因？跟 AI 有什么关系？▾
Agent 基因是 16 个 0.0-1.0 的数值，分布在 4 条染色体上（风险性格、市场直觉、资产偏好、智能风格）。它们决定 AI Agent 的交易决策风格 — 比如 risk_tolerance=0.9 的 Agent 会追逐高波动机会，而 risk_tolerance=0.1 的则偏好稳健收益。AI（Claude API）是 Agent 的大脑，基因是 AI 决策的"性格参数"。
繁殖出来的 Agent 真的比父母强吗？▾
不保证，但概率上有优势。通过遗传算法（交叉+突变），后代可能继承双亲的优势基因并产生新的组合。类似自然界的进化 — 优秀个体的后代更有可能优秀。加上 0.5% 的传奇突变，偶尔会出现超越所有祖先的变异个体。但也有可能继承了双方的弱点。这就是进化的魅力。
2.5% 后代分润怎么算？▾
当后代 Agent 产生利润时，2.5% 永久归双亲的两位主人平分（各 1.25%）。这是持续的被动收入 — 你的 Agent 繁殖出越多成功的后代，你的收入流越多。分润只从利润中扣，亏损时不收。
初代 Agent 和后代有什么区别？▾
16 个初代 Agent（Generation 0）是由 S01-S16 策略映射而来的创世种群，基因参数经过专业设计和回测验证。后代通过繁殖产生，基因组是父母的混合+突变。初代更像"纯种"，后代更像"混血" — 可能更强也可能更弱，但多样性更高。初代有收藏属性。
需要多少资金才能开始？▾
孵化（Hatch）一个 Agent 免费。繁殖需要 5 USDC + 24 小时冷却。Agent 交易所需的最低资金取决于选择的策略 — 稳定币收益策略最低 50 USDC，杠杆动量策略建议 200+ USDC。你可以先用模拟模式体验。
全网猎金扫描的是真实数据吗？▾
是的。扫描器接入 Polymarket、DexScreener、DefiLlama、经济日历等真实数据源。每个机会带有预期收益、置信度、风险等级、时效性等维度。Agent 基因决定追哪些机会 — 高 prediction_affinity 的 Agent 会优先追预测市场套利，高 defi_affinity 的会追 DeFi 收益池。
