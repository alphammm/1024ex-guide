AI 不是印钞机

我们必须诚实地说：AI Agent 不保证盈利。即使是那个一周赚 $115K 的 Agent，也是利用了一个特定的、短暂的预言机延迟窗口。这类机会不会永远存在。

社区数据显示，大多数 Polymarket 交易者并不盈利，只有少数人保持持续正收益。AI Agent 提高了效率和速度，但不改变市场的零和本质 —— 你赚的，是别人亏的。

正确的心态是：把 AI Agent 当作信息处理和执行的加速器，而不是"躺赚神器"。设好风控（单笔限额、每日上限、紧急熔断），让 Agent 在规则内运行。

另一个风险：安全。很多用户跑 OpenClaw 时 API Key 配置不当、端口暴露。务必使用只读 + 交易权限的 API Key，永远不要给 Agent 提币权限。
