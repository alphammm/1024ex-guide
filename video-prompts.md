# 1024EX 宣传片 — Nano Banana Video Prompts

> 使用方法：打开 https://nanobanana.io/text-to-video → 选模型（推荐 Veo 3.1 或 Sora 2 Pro）→ 粘贴 Prompt → 设置参数 → 生成
>
> 通用建议设置：
> - 分辨率：1080p
> - 宽高比：16:9（横版）或 9:16（竖版短视频）
> - 时长：10-15 秒

## 使用小贴士

1. **模型选择**：复杂场景用 Veo 3.1 或 Sora 2 Pro，简单场景用 Kling 2.6 更快
2. **多次生成**：同一 Prompt 建议生成 3-4 次，选最好的一条
3. **后期拼接**：可以把多条视频在剪映/CapCut 里拼接，加上 BGM 和文字
4. **文字叠加**：Nano Banana 生成的视频中文字可能不清晰，建议后期用剪辑工具叠加 1024EX Logo 和文案
5. **竖版优先**：如果投放 TikTok / 抖音 / Reels，用 9:16 比例生成
